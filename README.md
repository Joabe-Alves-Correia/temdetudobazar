# 🛍️ Tem de Tudo Bazar — Site Oficial

Site desenvolvido como projeto de extensão universitária para digitalização
de pequeno comércio local em Recife - PE.

## 🏪 Sobre a Loja

**Tem de Tudo Bazar** é uma loja de variedades localizada no coração de Recife,
especializada em papelaria, material escolar, utilidades domésticas e brinquedos.

- 📍 Av. Dantas Barreto, 645 - Santo Antônio, Recife - PE
- 📞 (81) 9 9957-7961
- 🕐 Segunda a Sábado: 8h às 18h
- ⭐ Avaliação Google: 4.4

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com variáveis e animações
- JavaScript puro (Vanilla JS)
- Font Awesome (ícones)
- Google Fonts
- Google Analytics
- Schema.org (dados estruturados)

## ♿ Acessibilidade

O site foi desenvolvido seguindo as diretrizes WCAG 2.1 com:

- Atributos ARIA em todos os elementos interativos
- Navegação completa por teclado
- Skip link para conteúdo principal
- Contraste adequado entre texto e fundo
- Textos alternativos em elementos visuais

## 🔍 SEO

- Meta tags Open Graph para redes sociais
- Schema.org LocalBusiness
- Sitemap.xml
- Robots.txt
- Semântica HTML5 correta

## ⚡ Funcionalidades

- Loader animado de entrada
- Navbar responsiva com menu mobile
- Parallax suave no hero
- Status de funcionamento em tempo real
- Filtro de produtos por categoria
- Sistema de favoritos (wishlist)
- Contador animado de estatísticas
- Countdown de promoção
- Formulário com validação e máscara de telefone
- Integração direta com WhatsApp
- Mapa do Google integrado
- Scroll reveal nas seções
- Toast notifications
- Botão voltar ao topo
- Banner de cookies
- Design 100% responsivo

## 🎓 Projeto de Extensão

Este projeto foi desenvolvido com o objetivo de promover a inclusão digital
de pequenos comerciantes locais, conectando a universidade à comunidade
e contribuindo para o desenvolvimento econômico do comércio local de Recife.

**Impacto Social:**
- Digitalização de pequeno comércio local
- Aumento de visibilidade online
- Facilitação do contato com clientes
- Democratização do acesso à tecnologia
